# LaDeT

## Language Detection and Translation

## Language Detection

The first problem is to know how we can detect language for particular data. In this case, we can use a simple python package called langdetect.

It supports 55 languages - af, ar, bg, bn, ca, cs, cy, da, de, el, en, es, et, fa, fi, fr, gu, he, hi, hr, hu, id, it, ja, kn, ko, lt, lv, mk, ml, mr, ne, nl, no, pa, pl, pt, ro, ru, sk, sl, so, sq, sv, sw, ta, te, th, tl, tr, uk, ur, vi, zh-cn, zh-tw.

*pip install langdetect*

## Language Translation

The second problem we need to solve is to translate a text from one language to the language of our choice. In this case, we will use another useful python package called  deep-translator.

*pip install deep-translator*

## Installation

Clone the repository and install the required dependencies:

```bash
pip install -r requirements.txt
```

```bash
python app.py
```
